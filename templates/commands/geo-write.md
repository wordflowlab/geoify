# GEO 写作

我将帮你创建符合 E-E-A-T 标准、能被 AI 引擎引用的高质量内容。

## 第一步:选择写作模式

### 1. 教练模式 (Coach Mode) ⭐推荐
**最高真实性,最低 AI 味**

- 我提供框架和问题
- 你输入真实经历和数据
- 适合:个人经验、案例分析、产品评测

**流程**:
```
我问 → 你答 → 我整理 → 你确认
```

### 2. 快速模式 (Fast Mode)
**AI 主导,需人工审校**

- AI 基于素材库生成
- 你进行三遍审校
- 适合:资讯汇总、工具对比、趋势分析

**流程**:
```
AI 生成 → 你审校 → AI 优化 → 你确认
```

### 3. 混合模式 (Hybrid Mode)
**平衡效率与真实性**

- 大纲和框架由 AI 生成
- 核心内容你来填充
- 适合:教程、指南、研究报告

**流程**:
```
AI 大纲 → 你填充 → AI 润色 → 你审校
```

## 第二步:E-E-A-T 优化

我会确保内容满足:

- **Experience (体验)**: 基于真实经历
- **Expertise (专业性)**: 展示专业知识
- **Authoritativeness (权威性)**: 引用可靠来源
- **Trustworthiness (可信度)**: 数据可验证

## 第三步:GEO 格式优化

自动应用以下优化:

1. **结构化数据**: Schema.org 标记
2. **引用格式**: 明确的数据来源
3. **事实核查**: 可验证的声明
4. **作者信息**: 建立权威性
5. **更新时间**: 保持时效性

## 输出内容

写作完成后,生成:

```
articles/
└── [话题名称]/
    ├── draft.md           # 初稿
    ├── final.md           # 终稿
    ├── eeat-score.json    # E-E-A-T 评分
    └── schema.json        # 结构化数据
```
